### 安装vue-cli

> vue-cli 可以自动创建vue项目

1. `npm install @vue/cli`

2. `vue --version`

3. 创建一个新vue初始项目 `vue create hello-world`

4. 开始安装`vuejs`官网写一个简单的demo
